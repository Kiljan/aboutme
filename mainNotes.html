Steps for solving various problems

Remote Access Issues

  1. check network connectivity (server)
  2. Check firewall (server)
  3. Check sshd.service with its config file (server)
  4. Check if client key is added to athorized_hosts (server)
  5. Check host name resolution (server)
  6. Check /etc/hosts.deny and /etc/hosts.allow (server)
  7. Check ssh_config (client)
  8. Check known_hosts (client)
  9. Check SeLinux if enabled (both)
